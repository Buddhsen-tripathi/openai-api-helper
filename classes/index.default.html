call

  • call(model, prompt, temperature?, resultType?): Promise<string | AxiosResponse<OpenAIAPIResponse, any>>

  • Makes a request to the OpenAI API with the provided parameters and returns the result.

    Parameters

    • model: string

      The GPT model to use (e.g., "gpt-3.5-turbo").

    • prompt: string

      The prompt text to send to the model for generating a response.

    • Optionaltemperature: number = 0.7

      (Optional) The sampling temperature, which controls the randomness of the model's output. Lower values make the output more deterministic.

    • OptionalresultType: number = 1

      (Optional) Determines the type of result returned. 0 returns the complete Axios response object, while 1 returns just the generated message content.

    Returns Promise<string | AxiosResponse<OpenAIAPIResponse, any>>

    • Returns the full Axios response if resultType is 0, or just the message content if resultType is 1.

    Throws

    Will throw an error if the API call fails.

    Example

    const apiHelper = new OpenAIAPIHelper('your-api-key');
    const message = await apiHelper.call('gpt-3.5-turbo', 'Tell me a joke', 0.5, 1);
    console.log(message);  // Outputs the joke generated by the model.
